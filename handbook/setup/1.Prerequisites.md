# Prerequisites

## Required Accounts & Access

- **GitHub Enterprise** organization with admin access
- **GitHub Copilot** subscription (active)
- **Azure DevOps** organization & project
- **VS Code** 1.90+ installed locally

## GitHub Setup

- [ ] GHAS enabled: Dependency Scanning, Secret Scanning, CodeQL
- [ ] GitHub Actions enabled
- [ ] GitHub PAT created (scopes: `repo`, `read:org`, `workflow`)
- [ ] Branch protection on `main` branch

## Azure DevOps Setup

- [ ] Azure DevOps PAT created (scopes: Work Items, Code, Build, Project)
- [ ] GitHub service connection configured

## VS Code Extensions

Install from Marketplace:

- GitHub Copilot
- GitHub Copilot Chat
- GitHub Pull Requests and Issues
- Azure DevOps
- Extension Pack for Java
- Language Support for Java(TM) by Red Hat
- Spring Boot Extension Pack
- ES7+ React/Redux/React-Native snippets
- REST Client

## Local Setup

- [ ] Java 17+ installed
- [ ] Maven 3.9+ installed
- [ ] Node.js 18+ installed
- [ ] npm 9+ available
- [ ] Git configured: `git config --global user.name` & `user.email`

## MCP Servers (.vscode/mcp.json)

Configure Azure DevOps and GitHub MCP servers:

- [Azure DevOps MCP server](https://github.com/mcp/microsoft/azure-devops-mcp)
- [GitHub MCP server](https://github.com/mcp/github/github-mcp-server)

Store PATs in VS Code input prompts, not in config files.

## Quick Verification

```bash
# Java
java --version    # Should be 17+

# Maven
mvn --version     # Should be 3.9+

# Node.js / npm
node --version    # Should be 18+
npm --version     # Should be 9+

# GitHub API
curl -H "Authorization: token $GITHUB_TOKEN" https://api.github.com/user

# Azure DevOps API
curl -u :<AZURE_DEVOPS_PAT> https://dev.azure.com/<org>/_apis/projects

# Backend
cd src/backend && mvn spring-boot:run       # Should start on port 8080

# Frontend
cd src/frontend && npm install && npm start # Should start on port 3000
```
